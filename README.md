# simple-interest-calculator

This project calculates simple interest based on the principal amount, interest rate, and time.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Example
Principal = 1000
Rate = 5%
Time = 2 years

Simple Interest = (1000 × 5 × 2) / 100 = 100
